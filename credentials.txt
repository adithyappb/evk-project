-------------------------------------------------
The real Gmail App Password is stored in the .env file (gitignored).
To run with email delivery enabled:
  1. Get the .env file from Chris via secure channel (Slack DM / iMessage).
  2. Place it at: EVkids code/evk-project/.env
  3. Run: cd "EVkids code/evk-project" && uv run evk serve

Without .env the app runs in local/stub mode — fully functional for development,
OTP codes appear in the browser banner instead of being emailed.

Running the server
------------------
  cd "EVkids code/evk-project"
  uv run evk serve

Running the pipeline manually
------------------------------
  uv run evk poll       # fetch unread emails from Gmail inbox
  uv run evk digest     # match opportunities → generate draft emails
  uv run evk remind     # send deadline reminders

Running tests
-------------
  uv run python -m pytest tests/ -q
