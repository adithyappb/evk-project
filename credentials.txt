── Running the server ───────────────────────────────────────────────────────

  cd "EVkids code/evk-project"
  uv run evk serve

  Gmail inbox polls automatically once daily (AUTO_POLL=true, 1440 min).
  Manual triggers:
    uv run evk poll       # poll Gmail inbox now
    uv run evk remind     # send deadline reminders now
    uv run evk digest     # build weekly digest drafts

  Tests:
    uv run python -m pytest tests/ -q
